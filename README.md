This is a **Virtual Reality maze simulation** developed using **Unity**.  
The environment is a **custom-designed maze modeled in Blender** and integrated into Unity for an immersive VR experience.

The application is built specifically for **VR devices** and allows users to navigate through the maze in virtual reality.

##  3D Assets
- The maze environment was fully modeled using **Blender**
- The Blender maze model is included in this repository

##  APK (VR Build)
- The final **VR APK** is available in the **Releases** section
- Install the APK on a compatible **VR device**
- Launch the app in VR mode to experience the maze

##  Tools & Technologies
- Unity
- Blender (3D modeling)
- VR SDK (Unity XR)
- Android (VR APK)

##  Requirements
- Android-based VR device (e.g., VR headset)
- Motion controller or headset-based navigation (depending on device)
